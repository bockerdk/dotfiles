* Use-package
Convenience macro for installing and configuring packages

Require hooks to be typed out in full

#+begin_src emacs-lisp
  (use-package use-package
    :custom
    (use-package-hook-name-suffix nil))
#+end_src

Never install a package unless explicitly stated (~:ensure t~)

#+begin_src emacs-lisp
  (use-package use-package
    :custom
    (use-package-always-ensure nil))
#+end_src
